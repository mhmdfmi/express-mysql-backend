Express MySQL Backend API

A secure, scalable RESTful API for user & product management with JWT, API key, and image upload support.

Dark/Light Mode Responsive API Key Protected

Authentication

Register

POST /api/v1/users/auth/register 
Headers:
  api-key: <your-api-key>
  Content-Type: application/json

Body:
{
  "name": "John Doe",
  "email": "john@example.com",
  "password": "password123"
}
Register a new user. Returns user data (without password).

Login

POST /api/v1/users/auth/login 
Headers:
  api-key: <your-api-key>
  Content-Type: application/json

Body:
{
  "email": "john@example.com",
  "password": "password123"
}
Login and get JWT token for protected endpoints.
Response:
{
  "statusCode": 200,
  "message": "Login successful",
  "data": { "token": "..." }
}

Users

Get Users (Paginated)

GET /api/v1/users?page=1&pageSize=20 
Headers:
  api-key: <your-api-key>
  Authorization: Bearer <token>
Get paginated users. Requires JWT token.

Get All Users

GET /api/v1/users/all 
Headers:
  api-key: <your-api-key>
  Authorization: Bearer <token>
Get all users (no pagination). Requires JWT token.

Update Profile

PUT /api/v1/users/me 
Headers:
  api-key: <your-api-key>
  Authorization: Bearer <token>
  Content-Type: application/json

Body:
{
  "name": "New Name",
  "email": "new@email.com",
  "password": "newpassword"
}
Update your own profile. Only provided fields will be updated.

Delete Account

DELETE /api/v1/users/me 
Headers:
  api-key: <your-api-key>
  Authorization: Bearer <token>
Delete your own account.

Products

Get Products (Paginated)

GET /api/v1/products?page=1&pageSize=10 
Headers:
  api-key: <your-api-key>
Get paginated products.

Get All Products

GET /api/v1/products/all 
Headers:
  api-key: <your-api-key>
Get all products (no pagination).

Get Product by ID

GET /api/v1/products/1 
Headers:
  api-key: <your-api-key>
Get product detail by ID.

Create Product (with Image)

POST /api/v1/products 
Headers:
  api-key: <your-api-key>
  Authorization: Bearer <token>
  Content-Type: multipart/form-data

Fields:
  name: "Product Name"
  price: "1000.00"
  description: "Description"
  image: (file, optional)
Create a new product. Note: Maximum image size is 1 MB.

Update Product (with Image)

PUT /api/v1/products/1 
Headers:
  api-key: <your-api-key>
  Authorization: Bearer <token>
  Content-Type: multipart/form-data

Fields:
  name: "New Name"
  price: "2000.00"
  description: "Updated description"
  image: (file, optional)
Update product by ID. Note: Maximum image size is 1 MB.

Delete Product

DELETE /api/v1/products/1 
Headers:
  api-key: <your-api-key>
  Authorization: Bearer <token>
Delete product by ID.

API Key Usage

How to Use

GET /api/v1/products 
Headers:
  api-key: <your-api-key>

or

GET /api/v1/products?api_key=<your-api-key>
All endpoints require a valid API key, either as a header or query parameter.

Frontend Usage Examples

Fetch Products with fetch() (Vanilla JS)


// Fetch products with API key
fetch('http://localhost:3001/api/v1/products', {
  headers: { "api-key": "<your-api-key>" }
})
  .then(res => res.json())
  .then(data => {
    // Loop products and show image
    data.data.forEach(product => {
      const img = document.createElement('img');
      img.src = product.image_url || '/uploads/default.jpg';
      img.width = 200;
      img.height = 200;
      img.style.objectFit = "cover";
      img.style.borderRadius = "0.5rem";
      document.body.appendChild(img);
    });
  });
This example fetches products and displays their images on the page.

Fetch Products with React/Next.js


// Example: pages/products.js (Next.js)
import { useEffect, useState } from "react";

export default function Products() {
  const [products, setProducts] = useState([]);
  useEffect(() => {
    fetch("http://localhost:3001/api/v1/products", {
      headers: { "api-key": "<your-api-key>" }
    })
      .then(res => res.json())
      .then(data => setProducts(data.data));
  }, []);
  return (
    <div style={{display:"flex",flexWrap:"wrap",gap:"1rem"}}>
      {products.map(product => (
        <div key={product.id} style={{
          background:"#1e293b",padding:"1.2rem",borderRadius:"0.5rem",boxShadow:"0 2px 12px rgba(0,0,0,0.12)",width:"16rem"
        }}>
          <img
            src={product.image_url || "/uploads/default.jpg"}
            alt={product.name}
            width={200}
            height={200}
            style={{ objectFit: "cover", borderRadius: "0.5rem" }}
          />
          <div style={{marginTop:"0.5rem",fontWeight:"bold"}}>{product.name}</div>
          <div style={{fontSize:"0.95rem",color:"#94a3b8"}}>{product.description}</div>
          <div style={{color:"#3b82f6",marginTop:"0.25rem",fontWeight:"600"}}>{product.price}</div>
        </div>
      ))}
    </div>
  );
}
This React/Next.js example fetches and displays products with images and details.

Upload Product Image with Postman

Headers:
  api-key: <your-api-key>
  Authorization: Bearer <token>
  Content-Type: multipart/form-data

Body (form-data):
  name: "Product Name"
  price: "1000.00"
  description: "Description"
  image: (choose file)
Use Postman form-data, add your API key and JWT token in headers.

Get Product Image in Frontend

<img src="http://localhost:3001/uploads/your-image.jpg" alt="Product Image" width="200" height="200" style="object-fit:cover; border-radius:0.5rem;" />
Use the image_url field from API response for product images.
